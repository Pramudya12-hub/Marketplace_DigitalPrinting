# MarketPlace Digital Printing 

Project ini merupakan aplikasi sederhana berbasis python yang menerapkan konsep Object Oriented Programming dan menggunakan pustaka library python yaitu Tkinter sebagai Graphical User Interface (GUI). 

Aplikasi ini memungkinkan pengguna untuk melihat produk-produk digital printing, menambahkan produk ke keranjang, menghapus produk dari keranjang, hingga checkout produk

# Fitur utama 
- Menampilkan daftar produk (Banner, PhotoPrint, Merchandise)
- Menambahkan produk ke keranjang
- Menghapus item dari keranjang
- Perhitungan total harga otomatis
- Diskon berbeda untuk setiap kategori produk (polymorphism)
- Antarmuka GUI sederhana dan mudah digunakan

## Teknologi yang Digunakan
- Python 3.x  
- Tkinter (GUI)  
- Konsep OOP: Inheritance, Encapsulation, Polymorphism
